## 💡 测试技巧

1. **打开开发者工具**
   - F12 或右键 → 检查
   - 查看 Console 是否有错误

2. **测试网络问题**
   - DevTools → Network → 限制网速
   - 测试慢网络场景

3. **清除缓存**
   - DevTools → Application → Clear site data
   - 测试首次加载

4. **模拟移动设备**
   - DevTools → Toggle device toolbar
   - 测试响应式布局
